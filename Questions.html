Questions For Class

What is meant by "nested" HTML elements?

One element can be placed inside another element.

What are the main two different types of HTML List elements?

There is a ordered list and an unordered list.

What is the purpose of the "href" HTML attribute for an anchor tag?

It specifies the URL of the page the link goes to.

What is the purpose of the "src" HTML attribute for an image tag?

It specifies the path to the image to be displayed.

Which HTML attribute can be used to add cascading style sheets (CSS)?

The style tag.
